## ⬇️ Downloads

Visitors can download the resume directly from the page in **two formats**:

| Format | Button on page | What opens |
|---|---|---|
| **Word (.docx)** | `Download .docx` (filled blue button) | Opens Save dialog — no internet needed, file is embedded |
| **PDF** | `Download PDF` (outlined blue button) | Generates A4 PDF client-side via `html2pdf.js`, then downloads |

---

## 🛠️ Built With

| Tech | Purpose |
|---|---|
| **HTML5 / CSS3 / Vanilla JS** | Entire site — zero frameworks, zero dependencies |
| **Google Fonts** | Fraunces (display) · DM Mono (code labels) · Outfit (body) |
| **html2pdf.js** (cdnjs) | Client-side PDF generation from the resume content |
| **Base64 embedding** | Profile photo & .docx file embedded directly — no extra files needed |
| **GitHub Pages** | Free static hosting, live at `dipak491.github.io/Resume` |

## 🗂️ Repo Structure

```
Resume/
└── index.html    ← The entire site (self-contained single file)
```

> Everything — photo, fonts fallback, resume content, and downloadable `.docx` — is embedded inside `index.html`. No folders, no assets, no server needed.

## 🚀 How It's Deployed

1. Repo is set to **GitHub Pages** → `main` branch → `/ (root)`
2. `index.html` at root is automatically served as the homepage
3. Any push to `main` goes live in ~30 seconds
